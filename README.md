# Sudoku Solver & Algorithm Visualizer

## Overview

An interactive Sudoku Solver developed using Python and Tkinter that solves any valid 9×9 Sudoku puzzle using the recursive backtracking algorithm. The application also includes a visualization mode that demonstrates the solving process step-by-step.

## Features

- Interactive 9×9 Sudoku board
- Recursive backtracking algorithm
- Instant solve mode
- Algorithm visualization mode
- Input validation
- Clear board functionality

## Technologies Used

- Python
- Tkinter
- Recursive Backtracking
- Object-Oriented Programming
